### 核心设计原则

#### Beginner Level
1. **用户体验设计 (UX)**
   - 什么是 UX 设计
   - UX 设计的重要性
   - 以用户为中心的设计

2. **用户界面设计 (UI)**
   - 什么是 UI 设计
   - UI 设计最佳实践
   - 移动端设计

3. **视觉设计基础**
   - 字体排版 (Typography)
   - 色彩理论 (Color Theory)
   - 布局与构图 (Layout and Composition)

#### Intermediate Level
1. **用户研究**
   - 用户画像 (User Personas)
   - 用户旅程地图 (User Journey Mapping)
   - 可用性测试 (Usability Testing)

2. **信息架构与交互设计**
   - 信息架构
   - 交互设计
   - 微交互

3. **原型设计**
   - 线框图 (Wireframing)
   - 高保真原型

#### Expert Level
1. **高级设计技术**
   - 无障碍设计 (Accessibility)
   - 国际化设计
   - 设计系统 (Design Systems)

### 快速行动指南
当需要调整 UI 时：
1. 先了解当前 UI 结构
2. 应用视觉设计基础原则（字体、色彩、布局）
3. 考虑用户体验和交互
4. 遵循 UI 设计最佳实践
5. 测试和迭代
